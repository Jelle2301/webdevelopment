Antwoorden Demo's


Demo 1 globale variabelen.

Blijft de data behouden als je de pagina herlaadt?

De data wordt niet behouden wanneer we de pagina refreshen.

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

De data wordt hier ook niet behouden.

Gebruiken twee tabbladen die dezelfde pagina tonen, dezelfde data?

Nee ze delen de data niet, de data wordt beschouwd als 2 verschillen.


Demo 2 session storage

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

Data wordt niet begehouden omdat we onze sessie gesloten hadden.

Blijft de data behouden als je de pagina herlaadt?

Data wordt bijgehouden omdat we onze sessie open houden.

Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?

Nee de data wordt apart bekeken, dit ook omdat we een andere sessie openen.

Denk je dat een andere pagina (andere file of url) aan de data geraakt als je ze in hetzelfde tabblad opent?

Nee dit ook omdat het telkens andere sessies zijn.


Demo 3 local storage

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?

Ja de data wordt behouden doordat we alles opslaan op de computer/laptop.

Blijft de data behouden als je de pagina herlaadt?

Data wordt behouden, zelfde reden als hierboven.

Blijft de data behouden als je de browser afsluit, opnieuw start en de pagina weer opent?

Data wordt behouden, weer zelfde reden als hierboven.

Blijft de data behouden als je de pagina in een andere browser opent?

Nee data wordt bijgehouden op de pc/laptop maar ook per browser.

Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?

Ja data wordt gedeeld.

Denk je dat een andere pagina (andere file of url) aan de data geraakt?

Neen
